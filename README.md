# RenamePdfPages

Программа для автоматического переименования PDF файлов с добавлением количества страниц в название.

## Возможности

- Поиск PDF файлов в выбранной папке
- Подсчет количества страниц каждого PDF
- Автоматическое переименование файлов

Пример:

До:
document.pdf

После:
document л. - 15.pdf

где `15` — количество страниц.


## Требования

- Windows 10/11
- .NET 8 (для запуска из исходного кода)


## Используемые библиотеки

- UglyToad.PdfPig


## Запуск

Запустить проект:
dotnet run

После запуска указать путь к папке с PDF файлами.


## Создание standalone EXE

Команда:


dotnet publish -c Release

Готовый файл будет находиться:
RenamePdfPages
│
├── RenamePdfPages.csproj
├── Program.cs
├── PdfRenamer.cs
├── .gitignore
└── README.md
